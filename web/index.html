Features

Everything you need.
Nothing you don't.

A feature-light, no-bullshit alternative to Postman and equivalents. Built by a developer who got tired of paying for features nobody asked for.

01

Request Builder

All HTTP methods. Full control over headers, query params, and body — JSON, XML, SPARQL, or raw. Environment variables with {{VARIABLE_NAME}} syntax.

02

Response Viewer

Status, time and size at a glance. Syntax-highlighted body. Click any JSON path to copy it as an inject reference. Headers and cookies included.

03

HTTP Trace

Full timeline of every connection phase — DNS lookup, TCP handshake, TLS negotiation, TTFB, transfer. Redirect chains tracked. Know exactly where every millisecond goes. Everything is calculated in local.

04

Dependency Graph

Chain requests. Inject values from any previous response into later fields using {{@file|body.id}} syntax. Run your whole collection in one click, in dependency order, with a live graph view.

05

Local-first & Git-ready

Requests are plain .postier JSON files in folders you own. No cloud sync, no account, no lock-in. Commit them to git, share them with your team, diff them like any other code.

06

Env variables & Secret Masking

Inject any variable across request by collection with {{base_url}} syntax. API keys, tokens, and passwords are auto-masked in the UI.

HTTP Trace

See every millisecond
of your request.

Postier hooks into Go's native HTTP transport to give you a full trace of every connection step. DNS lookup, TCP connect, TLS handshake, first byte, full transfer — with timestamps accurate to the millisecond.

Redirect chains are tracked and shown separately. Certificate details exposed on TLS connections. Never wonder where latency comes from again.

Dependency Graph

Chain requests.
Build real flows.

Use {{@filename|body[0].id}} to inject a value from any previous response into any field of a later request. Works on body (with full dot/bracket JSON path traversal), headers, and status codes.

Hit "Run all" and Postier resolves the dependency order automatically — then renders a live graph while it executes. Two independent chains are colored differently. Inspect every step.

Themes

Looks the way
you want it to.

Postier ships with Catppuccin and Rosé Pine out of the box. Don't like them? Drop a JSON file in the config folder and reload — no restart, no build step.

Your theme lives on disk like everything else. Version-control it, share it with teammates, or keep it private. It's just a file.
